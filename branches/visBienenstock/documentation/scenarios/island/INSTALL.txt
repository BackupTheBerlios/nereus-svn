Anweisungen zum installieren des IslandSzenarios

1. kopieren des servermoduls in den server
    cp -r servermodule/ ../../server/scenario/island/

2. kopieren des environments in den server
    cp -r environment/ ../../serer/scenario/islandenvironments

3. kopieren der agents in den client 
    cp -r agents/ <Pfad zum Client>/client/scenario/island

4. kopieren der communication in den client
   cp -r communication/ <Pfad zum Client>/client/scenario/communication
